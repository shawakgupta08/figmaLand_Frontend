# figmaLand_Frontend
 #  Frontend Landing Page Website

A fully responsive and interactive landing page built as part of the **JTG Frontend Home Assignment**.  
This project demonstrates strong skills in **HTML, CSS, and JavaScript**, along with UI/UX attention to detail such as animations, sliders, modals, video interaction, pricing UI, and contact form behavior.

---

---

## 📌 Features

### 🔹 Fully Responsive Layout  
Works across desktop, tablet, and mobile screens.

### 🔹 Hero Section  
Beautiful hero banner with curved clipping path and CTA buttons.  
*Source: index.html* :contentReference[oaicite:0]{index=0}

### 🔹 Features Section  
Grid-based feature cards with icons and clean typography.

### 🔹 Video Player with Play Overlay  
Custom interactive video play button and overlay.
*Handled in script.js* :contentReference[oaicite:1]{index=1}

### 🔹 Content Sections  
- MacBook mockup frame with embedded image  
- Newsletter subscription UI  
- Professional section headings & formatting

### 🔹 Partners / Clients Section  
Grid display of partner logos with borders and alignment.

### 🔹 Testimonials Slider (Carousel)  
Autoplay slider with:
- Dynamic slide creation  
- Dots navigation  
- Pause/play on interaction  
*Implemented in script.js* :contentReference[oaicite:2]{index=2}

### 🔹 Pricing Plans  
Three pricing cards with hover animation and highlighted plan.

### 🔹 Contact Form + Modal Popup  
Form submission triggers a success modal (no backend required).  
*Modal logic in script.js* :contentReference[oaicite:3]{index=3}

### 🔹 Embedded Google Map  
Integrated map with contact details & social icons.

### 🔹 Footer  
Multi-column footer with links and contact details.
